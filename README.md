## ALGO CAGE · Data Oracle

**EN**

This repository contains a single-page fictional web prototype for the project **“Algorithm Cage – Being ‘Fortune-Told’ by Big Data”**, created for  
*The University of Hong Kong, Master of Arts in Creative Communications, course code ENGL7512*.

The `index.html` file is a self-contained site (pure HTML/CSS/JavaScript) that simulates a behavioural prediction console.  
Visitors answer a few lightweight questions, then watch an “algorithm” generate:

- an internal subject code and “data shadow”
- risk and opportunity scores for the next 30 days
- a short behavioural forecast, written in the tone of a commercial data dashboard

The goal is not accuracy, but to show how little input is needed for a system to **pretend** it knows who you are.

**How to view**

- Locally: open `index.html` in any modern browser  
- Online: visit the GitHub Pages deployment for this repository (if enabled)

**中文说明（ZH）**

这个仓库收录的是一个单页网页原型，属于项目  
**《Algorithm Cage｜被大数据“算命”的人生》**，为  
**香港大学创意传播文学硕士（MA in Creative Communications），课程 ENGL7512 的课程项目**。

`index.html` 是一个完全本地运行的页面（纯 HTML/CSS/JavaScript），模拟的是一个「行为预测控制台」。  
观众只需回答几个看似无害的小问题，系统就会生成：

- 一个内部用户代号与「数据阴影」画像  
- 未来 30 天的风险与“机会”评分  
- 一段带有商业数据看板口吻的行为预测文案  

这个页面并不追求预测准确，而是想让人感受到：在真实世界里，系统往往只需要非常少的输入，就足以**假装**了解你是谁。

**使用方式**

- 本地预览：直接用任意现代浏览器打开 `index.html`  
- 在线浏览：访问本仓库对应的 GitHub Pages 页面（如已开启）
